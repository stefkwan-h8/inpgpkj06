Inline CSS

CSS bisa ditulis didalam sebuah halaman HTML, tidah harus dipisah ke file berbeda. Tapi ini tidak di rekomendasikan, karena kita jadi harus edit CSS di file HTML, kalau CSS panjang file nya jadi panjang juga. kurang rapi.

caranya adalah menaruh tag style di dalam head

Inline Javascript

JS juga bisa ditulis di dalam sebuah halaman HTML. sama, tidak di rekomendasikan juga, kurang rapi. caranya adalah dengan menggunakan tag script di dalam body

biasanya kita ambil sebuah elemen HTML menggunakan Javascript kemudian kita ubah ubah
